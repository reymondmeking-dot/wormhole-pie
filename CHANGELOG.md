# 更新日志 / Changelog

## [Unreleased]

### 改进

- 新增 Windows、macOS 与前端三路持续质量门禁，在合并前执行类型检查、生产构建、Rust 格式、Clippy 和测试。

## [0.2.0] - 2026-07-19

### 新增

- 完整的中英文界面、窗口标题、宠物右键菜单与系统托盘同步。
- 本地助手、Codex、Claude Code 与 Hermes 的独立持久化对话会话。
- 小红书、X、抖音托管 Edge Profile，以及仅聚合页面可见数据的社交中心。
- 指标驱动与手动选择并存的宠物进化路线。
- Blender 5.1 三维源资产、骨骼动画和透明 PNG 逐帧运行时资产管线。
- 公共桌面项目的受控 UAC 整理流程，以及不可迁移系统图标的可逆视觉收纳。

### 改进

- 强化文件拖放、资料库撤回、跨卷移动校验和 Windows 回收站撤销逻辑。
- 增强 Agent 安装、配置、任务隔离、附件快照与结果文件安全边界。
- 增加 Windows NSIS/MSI 和 macOS Universal DMG 的 GitHub Release 自动构建。

### 安全与隐私

- 应用不提取、复制或保存浏览器原始 Cookie；长期登录由隔离的浏览器 Profile 管理。
- 社交连接器不读取私信正文，仅返回受支持页面中可见的账号名称和汇总数字。
- 公共桌面整理必须显式确认并逐次提权，开发版或可写安装目录中默认禁用。
